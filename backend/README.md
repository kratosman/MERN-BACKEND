# MERN BACKEND

## Installation

Use the package manager to install .

```bash
npm install fs-extra
npm install mongoose
npm install express
npm install multer
npm install jsonwebtoken
npm install bcript
npm install cors
```

## Usage

```javascript
const express = require("express");
const mongoose = require("mongoose");
const app = express();
const cors = require("cors");
```
